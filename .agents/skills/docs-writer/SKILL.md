# 📝 Docs Writer (Ultra+)

🎯 **Purpose**
Generowanie i aktualizacja dokumentacji technicznej, API i manuali.

🛠️ **Implementation Logic**
1. **Scope**: Analiza kodu źródłowego (`src/` / `api/`).
2. **Format**: Czysty Markdown (GFM). Linkowanie względne plików.
3. **Structure**:
   - **Overview**: Krótki opis modułu/API.
   - **Interface/API**: Typy, parametry wejściowe, odpowiedzi (z typami).
   - **Examples**: Praktyczne przykłady użycia.

🗣️ **Usage Rule**
Zapisz zmiany w odpowiednim pliku `README.md`, `API.md` lub katalogu `docs/`.
